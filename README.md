# 📊 Finance Dashboard UI

A clean and interactive Finance Dashboard built using HTML, CSS, and JavaScript.
This project helps users track financial activity, manage transactions, and gain insights into spending patterns.

🚀 Live Demo




https://github.com/user-attachments/assets/8f955ff0-2ae1-493d-907d-179b55d40c36


📌 Features
💰 Dashboard Overview
Displays Total Balance, Income, and Expenses
Auto-calculated from transaction data
📋 Transactions Management
View all transactions with:
Date
Description
Amount
Category
Type (Income/Expense)
🔎 Search & Filter
Search transactions instantly
Filter by income or expense
🔐 Role-Based UI (Frontend Simulation)
Viewer
Can only view data
Admin
Can add new transactions
📈 Insights
Shows top spending category
Helps understand spending behavior
📱 Responsive Design
Works across mobile, tablet, and desktop
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
🧠 Approach

This project focuses on building a structured frontend without frameworks.

Key concepts used:

DOM manipulation
State handling using JavaScript
Modular and clean UI structure
Responsive layout design
